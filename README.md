# ansible-yarn

An Ansible role for installing Yarn.

Note that Yarn requires Node.js, version 4.0.0 or greater.

## Role Variables

- `yarn_version` - Yarn version (default: `1.19.1`)
- `nodejs_version` - Node.js version (default: `12.11.1`)
